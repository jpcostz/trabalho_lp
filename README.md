Realização do Pré-Processamento de C

## Descrição

Com a aplicação de funções que desempenham diversas partes do préprocessador, o arquivo precomdefine.c realiza a execução do pré-processamento entregando como saída o arquivo como um compilador entregaria.

## Execução do Programa

Para executar o programa, ao iniciar o arquivo .exe, com o .c que irá passar pelo pré-processamento no mesmo diretório, insira o nome do arquivo.

Exemplo com um arquivo de entrada chamado input e de saída chamado output: 

```bash
./pre.exe input.c output.
